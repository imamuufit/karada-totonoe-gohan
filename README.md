# imamu. からだ整えごはん

LINE公式アカウントのリッチメニューから開く想定の、スマホ優先の栄養・レシピ提案PWAです。

## 目的

医療診断ではなく、ユーザーが選んだ目的、最近の不調、食事傾向から、食事の見直しポイント、意識したい栄養素の候補、食材、現実的なメニュー案をその場で表示します。

## 主な機能

- 目的別モード選択
- 減量サポートの推定維持カロリー、減量開始カロリー、PFC目安
- 増量サポートの推定維持カロリー、増量開始カロリー、PFC目安
- 体調・不調、筋トレ回復、忙しい日向けの入力切り替え
- 安全確認カード
- プライバシー表記
- 体重管理PWAへのリンク導線
- ざっくりPFC早見表
- かんたん表示、くわしく表示の切り替え
- 今日の見直しポイント表示
- 意識したい栄養素表示
- おすすめ食材表示
- 自炊メニュー、コンビニ、外食時の選び方
- 避けすぎ注意ポイント
- LINE相談用コピー文
- リアルイマムへのLINE相談導線
- PWA manifestとservice worker

## 保存について

この初期版ではデータ保存を行いません。選択内容からその場で提案を表示するだけです。

## 注意

このアプリは医療診断、治療、栄養療法を目的としたものではありません。強い不調や長引く症状、妊娠中、治療中、医師から食事制限を受けている方は、医師や専門家へ相談してください。
